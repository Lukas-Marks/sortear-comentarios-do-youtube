## 🚀 Funcionalidades

- Acessa automaticamente um vídeo do YouTube.
- Faz scroll para carregar todos os comentários.
- Captura autores e textos dos comentários.
- Filtra comentários que contenham uma hashtag ou palavra-chave definida pelo usuário.
- Realiza um sorteio entre os comentários filtrados.
- Exibe no console o autor sorteado.

---

## 📦 Instalação

### 1. Clone este repositório

git clone https://github.com/seu-usuario/sortear-comentarios-do-youtube.git

cd sortear-comentarios-do-youtube

### 2. Instale as dependências

npm install puppeteer readline

### 3. Certifique-se de ter o Node.js instalado

Versão LTS recomendada.

Verifique com:

node -v
npm -v

---

## ▶️ Uso

Execute o script com:

node index.js

Ao iniciar, o programa perguntará no terminal:

Digite o comentário/hashtag que deseja procurar:

Digite, por exemplo:

#BolaDaCopa
